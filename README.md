# Dawn-UI

> Vue Component Library for Better Web Developing.

## Links

* [Doc](https://vue-dawn.danielmlc.com)

## Install

```bash
npm install dawn-ui --save
```

## Get Started

```javascript
// node modules dependency
import Vue from 'vue';
import BaseUI from 'dawn-ui';

// base-ui style
import 'dawn-ui/dist/dawn-ui.css';

// use plugin
Vue.use(DawnUI);
```
